## Setup

### tech stack 
- Typescript
- Nextjs
- Supabase
- Prisma
- Tailwindcss


First, run the local development server:

```bash
pnpm i 

// add env variables in .env

pnpm prisma db push

pnpm dev

// build

pnpm build
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
